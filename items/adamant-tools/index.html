Adamant Tools

Current answer

The confirmed Adamant tool set contains a spade, watering can, pickaxe and logging axe.

Tool list and use

  • Adamant Spade — soil and farming work.
  • Adamant Watering Can — crop care.
  • Adamant Pickaxe — mining progression.
  • Adamant Logging Axe — woodcutting progression.

Progression note

The tools arrived with the broader Adamant tier in 0.12. Exact durability, damage and gathering-speed values should be checked in the current build rather than copied from an older patch.
