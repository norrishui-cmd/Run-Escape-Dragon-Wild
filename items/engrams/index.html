Engrams

Current answer

Engrams are used to access Nightmare Crucible challenges. They connect normal progression to the replayable Crucible loop and its Ascension Core rewards.

How the item fits

Official 0.12 information identifies Engrams as the access item but does not publish a universal drop-rate table. Obtain and use them according to the current in-game instructions, then prepare for the selected challenge.

Next step

Read the Nightmare Crucible guide before spending an Engram, and see Ascension Cores for the confirmed reward relationship.
