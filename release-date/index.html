Nivalis Nights Release Date

Nivalis Nights releases for PC on Steam on September 29, 2026. Check the confirmed date, developer and publisher details.

Quick answer: Nivalis Nights is scheduled to release on Steam on September 29, 2026. It is developed by ION LANDS and published by 505 Games.

The confirmed date

The official Steam listing shows September 29, 2026 as both the release date and planned unlock date. ION LANDS also announced that date with the June 2026 name change from Nivalis to Nivalis Nights, clarifying that this is the same game with a new title and logo.

What players should do

Wishlist the game on Steam to receive the platform notification. The current listing is for Windows PC; an official release-date trailer says the Epic Games Store version will arrive later, without giving a separate date. This page should be rechecked if either storefront changes.
