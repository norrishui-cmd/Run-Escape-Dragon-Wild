Dragonwilds Dedicated Server Hosting Options

Published from official information dated 2026-03-30; editorial status reviewed 2026-07-19.

What changed

Players can use the hosting partner, run a server at home or manage a cloud machine; each option changes cost, setup effort and backup responsibility.

Confirmed details

  • A 64-bit Windows or Linux system is required.
  • RAM guidance is 2 GB plus 1 GB per player, up to six players in the documented 0.11 scope.
  • The first instance uses UDP 7777, with later instances moving upward.

What this means for players

Partner hosting reduces manual setup, while home and cloud options require the owner to manage networking and backups. Visibility in the public list and successful joining are separate checks.

Player checklist

  1. Choose a hosting model before configuring the world.
  2. Forward the correct UDP port through firewall and router.
  3. Back up Savegames and match every client to the server build.
