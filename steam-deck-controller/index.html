Nivalis Nights Steam Deck and Controller

Current controller input and Steam Deck verification status for Nivalis Nights.

Quick answer: The PC requirements state “Controller or keyboard + mouse.” The current Steam listing does not display a Valve Verified or Playable rating in the information reviewed for this guide.

Controller answer

Both minimum and recommended notes mention controller or keyboard and mouse, so controller input is planned. The storefront does not expose the final button layout, remapping options or whether gameplay can be completed without switching to a pointer.

Steam Deck answer

Controller input alone does not prove Steam Deck compatibility. A useful Deck verdict requires Valve’s rating or release-build tests covering performance, text size, launcher behavior, input prompts and battery life. Until then, this page avoids a Verified claim.
