Base Building

Starter Base Layout

A good starter base is compact, readable, and quick to return to after gathering loops.

Layout Rules

  • Place storage next to crafting stations so resource dumping is fast.
  • Keep cooking and healing supplies close to the exit.
  • Leave room for smelting, upgrade stations, and future resource processing.
  • Build in a location with predictable nearby resources and manageable enemy pressure.
