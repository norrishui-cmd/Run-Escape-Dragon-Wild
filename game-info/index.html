Confirmed Scope to Cover

  • RuneScape: Dragonwilds is an open-world survival crafting game set in Ashenfall.
  • Steam lists the Early Access launch as April 15, 2025; Jagex has announced version 1.0 and consoles for September 15, 2026.
  • The guide should cover crafting, building, skills, magic, exploration, combat preparation, dungeons, and dragon encounters.
  • Specific stats, exact recipes, enemy values, and drop rates should only be added after official notes or verified gameplay.

Current build at a glance

The live Early Access game has expanded beyond its April 2025 launch with Fellhollow, Dowdun Reach and Umbral Sands. Current systems include Fishing, dedicated servers, skill caps up to 99, new crafting stations, Water Magic, Steam Achievements and several late-game combat activities.

The announced next milestone is September 15, 2026, when Jagex plans to release version 1.0, Scorned Wilderness and console versions. See the release tracker and update hub for details.
