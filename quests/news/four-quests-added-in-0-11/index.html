Four Dragonwilds Quests Added in Update 0.11

Published from official information dated 2026-03-31; editorial status reviewed 2026-07-19.

What changed

Update 0.11 added four named quests, giving players new story and progression routes beyond the earlier quest set.

Confirmed details

  • Black Knight’s Fortress was added.
  • Wanted! and Biohazard were added.
  • What’s Theirs Is Mine completed the four-quest list in the official notes.

What this means for players

Players returning after an older build should update before looking for these objectives. Quest availability still depends on normal in-game progression, so presence in patch notes is not a promise that every character sees all four immediately.

Player checklist

  1. Update the game and server first.
  2. Read active prerequisites and quest-log objectives.
  3. Use a dedicated stuck-point guide only when the exact trigger or marker fails.

Official source and status

This article summarizes confirmed information and does not add unannounced mechanics, drop rates or release promises. Future-facing features remain labelled as announced until they are live.
