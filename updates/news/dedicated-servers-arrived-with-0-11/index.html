Dedicated Servers Arrived with Dragonwilds 0.11

Published from official information dated 2026-03-30; editorial status reviewed 2026-07-19.

What changed

Dedicated servers are live functionality with a documented setup path, not a feature still waiting for the 1.0 release.

Confirmed details

  • Windows and Linux 64-bit hosts are supported.
  • The documented capacity is up to six players in the 0.11 guide.
  • World owners can use partner, home or cloud hosting.

What this means for players

Older roadmap articles may still say dedicated servers were planned for later. The March 2026 setup documentation supersedes that timeline and should be the source used by current guides.

Player checklist

  1. Install the free dedicated-server product.
  2. Set mandatory owner, server, world and admin values while stopped.
  3. Match versions and maintain manual backups on self-hosted worlds.
