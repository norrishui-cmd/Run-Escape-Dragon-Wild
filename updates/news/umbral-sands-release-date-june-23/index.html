Umbral Sands Released on June 23, 2026

Published from official information dated 2026-06-11; editorial status reviewed 2026-07-19.

What changed

The announced June 23 release has passed, so Umbral Sands should be described as live content rather than an upcoming expansion.

Confirmed details

  • The region includes supernatural sandstorms, oases, Kalphites and Kot’Haar.
  • Fight Cave is replayable and leads toward Fuzan.
  • Magic Carpets provide a route over dunes and quicksand.

What this means for players

Search snippets and old previews may still use future tense. Current pages should distinguish the live 0.12 region from the later Scorned Wilderness and Kuldra release planned for September.

Player checklist

  1. Install the latest 0.12 hotfix, not only the base release.
  2. Follow current regional access requirements.
  3. Check the issue log when a quest, recipe or server behaves differently from launch-day coverage.
