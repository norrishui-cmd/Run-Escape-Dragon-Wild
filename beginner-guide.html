Beginner Guide

First-Day Route

Start with stability. A strong early game is built from tools, food, storage, shelter, and short scouting loops.

Opening Priorities

  1. Collect basic wood, stone, fiber, and nearby food before wandering far from spawn.
  2. Craft harvesting tools first so every minute spent gathering becomes more efficient.
  3. Build a compact starter base with storage, a workbench, cooking access, and a return point.
  4. Scout in short loops and return home often instead of carrying a full inventory into danger.
  5. Upgrade armor, food, and a ranged option before exploring ruins or lair-like areas.
