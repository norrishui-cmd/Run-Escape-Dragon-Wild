Map Guide

A strong map section should help players choose safe routes, locate resource loops, and decide where to build before pushing into harder areas.

Map Content Targets

  • Starter-safe exploration loops with return points.
  • Resource route notes connected to crafting progression.
  • Base location criteria: access, safety, nearby materials, and expansion room.
  • Landmark pages for ruins, lairs, bosses, and high-risk zones.
