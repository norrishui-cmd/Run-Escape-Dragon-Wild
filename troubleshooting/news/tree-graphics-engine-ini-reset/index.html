Official Engine.ini Reset for Distorted Trees

Published from official information dated 2026-06-23; editorial status reviewed 2026-07-19.

What changed

The official workaround recreates the local rendering configuration without deleting a character or world save.

Confirmed details

  • Close the game before changing the file.
  • The Windows path is %localappdata%\RSDragonwilds\Saved\Config\Windows.
  • Deleting Engine.ini makes the game generate a clean configuration on restart.

What this means for players

The reset can remove intentional custom graphics settings, so a backup is sensible. It should not be expanded into broad advice to delete the Saved directory, which also contains unrelated local data.

Player checklist

  1. Exit Dragonwilds completely.
  2. Back up and remove only Engine.ini from the documented folder.
  3. Restart, recheck graphics options and test the tree behavior.
