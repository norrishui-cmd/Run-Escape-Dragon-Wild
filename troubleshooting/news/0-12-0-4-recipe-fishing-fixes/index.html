Patch 0.12.0.4 Fixes Fishing and Adamant Bolts

Published from official information dated 2026-06-23; editorial status reviewed 2026-07-19.

What changed

Several apparent progression blocks were fixed together in 0.12.0.4, making version alignment the first diagnostic step.

Confirmed details

  • Fishing locations that could not be used were repaired.
  • Adamant Bolts no longer require crafting Adamant Arrows first.
  • Tok-Xil enemies correctly drop items on death.

What this means for players

A player on a current client can still encounter an old symptom when the dedicated server is behind. Check both sides before assuming the recipe, interaction or loot table has regressed.

Player checklist

  1. Compare full client and server versions.
  2. Restart after updating and revisit the interaction.
  3. Use the specialized station and current recipe before reporting a return of the bug.
