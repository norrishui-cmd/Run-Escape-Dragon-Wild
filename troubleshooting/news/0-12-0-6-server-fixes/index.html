Patch 0.12.0.6 Fixes Server Stations and Splinter

Published from official information dated 2026-06-23; editorial status reviewed 2026-07-19.

What changed

Dedicated-server owners should update to 0.12.0.6 or later before rebuilding missing stations or banning use of Splinter.

Confirmed details

  • Crafting and processing stations disappearing after server restarts were fixed.
  • The Splinter dedicated-server crash was fixed.
  • Fuzan leaving the Fight Cave arena and following respawned players was addressed.

What this means for players

Invisible collision could remain where a station appeared to vanish, so rebuilding immediately risks overlap and wasted materials. Patch, reload and inspect the original location first.

Player checklist

  1. Stop and back up the server.
  2. Update until the server log matches every client.
  3. Reload the world and inspect station locations before replacing anything.
