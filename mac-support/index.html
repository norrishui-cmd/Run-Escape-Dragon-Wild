Is Nivalis Nights Available on Mac?

Current macOS support status for Nivalis Nights.

Quick answer: No native macOS version is listed on the current official Steam or Epic Games Store pages. The published requirements specify 64-bit Windows.

Windows is the listed OS

Steam names Windows 10 and Windows 11 in both requirement tiers. The Epic listing also identifies Windows as its platform.

No compatibility promise

Running through translation, streaming or virtualization is different from native support and is not guaranteed by the developer. This page does not present those workarounds as an official Mac version.

What would confirm support

A macOS store badge, system-requirement tab or developer announcement is needed before the status changes.
