Crafting & Resources

Use crafting as a progression engine: better tools unlock better resources, and better resources unlock safer exploration.

Resource Page Structure

  • List each material by use case: tools, armor, cooking, stations, magic, or boss prep.
  • Explain when a resource becomes important, not only where it appears.
  • Group recipes by progression tier so players know what to save and what to spend.
  • Add farming loop notes for repeatable gathering routes and safe return paths.

Current top-tier material loop

Umbral Sands adds Adamant Ore, Yew Wood and Red Dragonhide as major progression materials. Adamant supports tools, melee and ranged equipment; Yew supports a fishing rod and ranged weapon components; Red Dragonhide can be tanned into armour. The 0.12 crafting-station rework also changes where players make this equipment.

Use the Adamant equipment list, Yew uses and station lookup before planning a long farming run.
