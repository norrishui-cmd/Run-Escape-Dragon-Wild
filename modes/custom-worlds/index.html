Custom World Settings

Current answer

Custom worlds let a host alter supported rule sliders instead of using only the default survival setup. The official Creative Mode update introduced this configuration direction.

Use cases

  • Builder-focused shared worlds.
  • Testing construction layouts.
  • Tailoring challenge rules for a regular group.
  • Separating experimental saves from a default progression world.

Host responsibility

Record the chosen rules for co-op players and keep backups before changing an established world. Available slider names and ranges can change between Early Access patches.
