Updates und Patchstände

Kurzantwort

Nutze immer die vollständige Versionsnummer. Kleine Hotfixes innerhalb von 0.12 können Rezepte, Erfolge, Angelstellen oder Serverfehler gezielt verändern.

Das Wichtigste

  • 0.12.0.4 korrigierte unter anderem Angelstellen und die Adamantbolzen-Voraussetzung.
  • 0.12.0.6 behob verschwindende Stationen und den Splinter-Serverabsturz.
  • Alte Guides benötigen deshalb einen sichtbaren Prüfstatus.

So gehst du weiter vor

  1. Vergleiche Client- und Serverbuild.
  2. Lies die Änderung für dein konkretes Problem.
  3. Sichere Serverwelten vor Updates.
  4. Teste die bisher fehlerhafte Aktion erneut.

Häufige Fragen

Warum ist die Unterversion wichtig?

Weil ein Server auf 0.12.0.1 nicht dieselben Korrekturen wie 0.12.0.6 enthält.
