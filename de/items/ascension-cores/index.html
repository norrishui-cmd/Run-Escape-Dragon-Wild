Ascension Cores

Kurzantwort

Ascension Cores werden über erfolgreiche Nightmare-Crucible-Herausforderungen verdient.

Bestätigte Daten

  • Ein Engramm gewährt den Aktivitätszugang.
  • Die Herausforderung muss erfolgreich abgeschlossen werden.
  • Eine garantierte Menge pro Lauf wird nicht erfunden.
