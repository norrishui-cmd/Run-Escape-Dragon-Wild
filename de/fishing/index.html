Angeln

Kurzantwort

Nicht nutzbare Angelstellen in Fellhollow und Brynmoor wurden mit Patch 0.12.0.4 behoben. Aktualisiere Client und Server und betrete das Gebiet anschließend neu.

Das Wichtigste

  • Nicht jede Wasserbewegung ist automatisch eine gültige Angelstelle.
  • Koop-Spieler und Server müssen denselben Build verwenden.
  • Ein alter Interaktionsfehler ist von falscher Ausrüstung zu unterscheiden.

So gehst du weiter vor

  1. Prüfe Version und Ausrüstung.
  2. Aktualisiere den dedizierten Server.
  3. Verlasse das Gebiet und kehre zurück.
  4. Teste eine eindeutig gültige Angelstelle.

Häufige Fragen

Warum reagiert die Stelle weiterhin nicht?

Prüfe, ob es wirklich eine Angelstelle ist und ob Methode sowie Ausrüstung zum Spot passen.
