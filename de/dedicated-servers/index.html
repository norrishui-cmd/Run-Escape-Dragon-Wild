Dedizierte Server

Kurzantwort

Installiere das kostenlose Serverprodukt, trage Owner-ID, Servername, Weltname und Admin-Passwort ein und halte Server sowie Clients exakt auf demselben Build.

Das Wichtigste

  • Benötigt werden 64-Bit-Windows oder Linux und etwa 2 GB Basis-RAM plus 1 GB pro Spieler.
  • Der erste Server verwendet UDP 7777; weitere Instanzen beginnen bei 7778.
  • Bis zu sechs Spieler werden unterstützt.

So gehst du weiter vor

  1. Stoppe den Server vor Konfigurationsänderungen.
  2. Sichere den Ordner Saved/Savegames.
  3. Öffne den benötigten UDP-Port in Firewall und Router.
  4. Prüfe die Version oben im Log.

Häufige Fragen

Warum erscheint mein Server nicht?

Prüfe Versionsgleichheit, exakte Großschreibung des Weltnamens, Pflichtfelder und anschließend das Serverlog.
