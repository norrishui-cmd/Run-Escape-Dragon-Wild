Fähigkeiten und Builds

Kurzantwort

Ein guter Build unterstützt eine konkrete Aufgabe. Kombiniere Überlebensfähigkeit, Ausdauer und passende Schadensoptionen, statt nur einzelne Maximalwerte zu verfolgen.

Das Wichtigste

  • Solo-Builds brauchen verlässliche Heilung und eine sichere Ausweichoption.
  • Im Koop können Rollen stärker spezialisiert werden.
  • Sammel-Builds sparen Zeit, ersetzen aber keine Vorbereitung auf Gebietsgefahren.

So gehst du weiter vor

  1. Lege Rolle und Zielinhalt fest.
  2. Wähle eine defensive Grundlage.
  3. Ergänze Schaden oder Sammeltempo.
  4. Teste Änderungen in einem bekannten Gebiet.

Häufige Fragen

Gibt es den einen besten Build?

Nein. Der beste Build hängt von Region, Gegner, Gruppengröße und verfügbarer Ausrüstung ab.
