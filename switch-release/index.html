Is Nivalis Nights Coming to Nintendo Switch?

Current Nintendo Switch and Switch 2 release status for Nivalis Nights.

Quick answer: No Nintendo Switch or Switch 2 version is currently announced by the developer or publisher. Official platform pages presently identify Windows via Steam and Epic.

Current status

Neither the current Steam listing nor the 505 Games Nivalis page displays a Nintendo platform badge. The game’s controller input support does not establish a Switch port.

Hardware considerations are not evidence

PC requirements call for 16 GB RAM and a dedicated GPU, but those specifications cannot be used to prove or disprove a custom console version. Only an official announcement can.

Avoid placeholder stores

Unverified retailer pages and search snippets will not be treated as confirmation. This page will update if Nintendo, ION LANDS or 505 Games publishes a listing.
