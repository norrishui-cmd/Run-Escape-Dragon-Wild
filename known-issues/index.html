Nivalis Nights Known Issues and Patch Notes

Current pre-release issue status and the future versioned troubleshooting policy.

Quick answer: Nivalis Nights has not released yet, so there is no public launch-build issue list. ION LANDS says it is optimizing performance and testing stability, saves, quests, UI, localization and compatibility.

Pre-release optimization work

The May update details LODs, light-probe optimization, NPC object pooling, reduced allocations and loading improvements. QA teams test general stability, save/load, quest progression, environments, characters, UI, memory and performance.

How issues will be documented

After launch, each issue needs a game version, affected platform or hardware, reproduction steps, workaround and official status. Old workarounds must remain tied to their version so fixed problems do not continue ranking as current advice.
