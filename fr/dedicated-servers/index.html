Serveurs dédiés

Réponse rapide

Installez le produit serveur gratuit, renseignez identifiant propriétaire, nom du serveur, monde par défaut et mot de passe administrateur, puis alignez exactement toutes les versions.

Points essentiels

  • Système Windows ou Linux 64 bits avec environ 2 Go de base plus 1 Go par joueur.
  • Le premier serveur utilise UDP 7777 ; les suivants commencent à 7778.
  • Le maximum publié est de six joueurs.

Comment continuer

  1. Arrêtez le serveur avant de modifier la configuration.
  2. Sauvegardez Saved/Savegames.
  3. Ouvrez le port UDP dans pare-feu et routeur.
  4. Contrôlez la version en haut du journal.

Questions fréquentes

Pourquoi le serveur n’apparaît-il pas ?

Vérifiez version, casse exacte du nom du monde, champs obligatoires puis journal du serveur.
