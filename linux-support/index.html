Is Nivalis Nights Available on Linux?

Current native Linux and SteamOS support status for Nivalis Nights.

Quick answer: No native Linux version is listed in the current official system requirements. Steam currently specifies 64-bit Windows 10 or 11.

Native support status

The absence of Linux from the OS list means it should not be marketed as a native Linux release. The Epic page likewise lists Windows.

Proton is a separate question

Community compatibility through Proton may be tested after release, but it is not equivalent to a developer-supported Linux build or a Valve Steam Deck rating.

Launch verification

Any Proton guide must record version, hardware, launch options, video playback, controller behavior and save compatibility before becoming indexable.
